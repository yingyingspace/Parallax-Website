視差捲動效果

作者：Vineet Kumar

什麼是視差捲動？

視差捲動是一種網頁設計技巧，網站背景的移動速度比前景更慢。當訪客往下捲動時，會產生 3D 效果，增加景深感並帶來更沉浸的瀏覽體驗。視差源自視覺錯覺。 由於人眼會將近處物體看得比遠處物體更大，我們會感覺遠處的物體移動得較慢。這種錯覺長期被 應用在各種媒介中，營造出更逼真的效果。它最早可追溯至傳統動畫，例如迪士尼《白雪公主》，以及 《超級瑪利歐》等遊戲。 隨著 CSS 與 HTML 的進步，視差效果逐步發展成為今日的網頁設計手法。

2007 年，網頁開發者 Glutnix 在部落格分享了最早的瀏覽器視差捲動實作之一，包含使用 JavaScript 與 CSS 2 的範例程式與示範，可支援當時的 Internet Explorer 6 等瀏覽器。然而直到 2011 年 HTML5 與 CSS 3 推出後，這項技術才受到網頁設計師廣泛採用。支持者認為它能展現網頁的 流動感，並藉由視差背景更好地吸引使用者、提升整體體驗。不過普渡大學於 2013 年的研究指出： 「……雖然視差捲動強化了某些使用者體驗面向，但不一定提升整體使用者體驗」。


為什麼要使用視差圖像？

    設計師應該嘗試視差圖像的兩個主要原因：

  • 有助於說故事

    視差圖像非常適合說故事。好的設計著重於溝通—設計網站時，我們希望與訪客交流，並傳達關於 產品、服務或品牌的故事。視差捲動能讓訪客沉浸在獨特的線上體驗中，甚至可以打造單頁網站， 讓訪客不必切換多個頁面就能看完整個品牌故事。

  • 有助於提升使用者參與

    視差圖像可用於提升使用者參與度。在網頁設計中，降低跳出率是重要目標之一。跳出率指的是 只看了一頁就按下「返回」或關閉分頁的訪客比例。高跳出率會影響搜尋排名，因此任何能降低 跳出率的措施都值得嘗試。精心設計的視差頁面能提供更有動態感的體驗，讓使用者願意停留更久。


若想在網站加入視差捲動效果，建議注意以下幾點：

  • 選擇性使用

    視差捲動在單一元素上使用會更有力，而不是每個頁面、每個區塊都使用。可以應用在頁首與標題， 或只用在首頁。目標是加強設計，而不是分散使用者注意力。

  • 圖片壓縮

    視差捲動會使用多種媒體素材與 CSS 圖形來營造深度感。不要讓過大的圖片檔拖慢載入速度， 影響網站效能與使用者體驗。

  • 善用色彩

    視差捲動不只是前景與背景的圖片配置，也與色彩有關。可以在背景使用較低飽和的色調， 前景使用更鮮明的色彩，以增加深度感。
